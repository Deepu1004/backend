└───────────────────────────────────────────────────────────────────────────┘

Problem: "CORS error" in browser
Solution: Ensure FRONTEND_URL environment variable is set correctly
         Include https:// and no trailing slashes

Problem: "Build failed" on Render
Solution: Check Render logs for specific error
         Common: ensure requirements.txt exists
         Try: pip install -r requirements.txt locally first

Problem: Backend URL not found
Solution: Render takes 2-5 minutes to deploy
         Watch build logs in Render Dashboard
         URL shows once deployment completes

Problem: File uploads not working
Solution: Check that BACKEND_URL is set correctly
         Verify frontend uses correct API URL
         Check browser Network tab for API errors

Full troubleshooting: See RENDER_DEPLOYMENT.md
