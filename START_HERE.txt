================================================================================
                      STEP 2: CREATE ON RENDER.COM
================================================================================

1. Go to: https://render.com
2. Click "New +"
3. Choose "Web Service"
4. Select "Deepu1004/backend" repository
5. Click "Connect"

✓ Repository connected to Render

================================================================================
                    STEP 3: CONFIGURE & SET VARIABLES
================================================================================

On the configuration page, set:

NAME:           authorprint-backend
BRANCH:         main
BUILD CMD:      $ pip install -r requirements.txt
START CMD:      $ uvicorn main:app --host 0.0.0.0 --port 8000
INSTANCE TYPE:  Free (or Starter for production)

Add Environment Variables:
  FRONTEND_URL  →  https://tnf-risk-analyzer.vercel.app
  PORT          →  8000
  DEBUG         →  False
  PYTHONUNBUFFERED → 1

LEAVE BACKEND_URL EMPTY FOR NOW (add after deploy gets URL)

Click "Create Web Service"

✓ Deployment started!

================================================================================
                   STEP 4: WAIT & GET YOUR BACKEND URL
================================================================================

Render will deploy automatically (2-5 minutes).

Once complete, your URL appears at top:
  https://authorprint-backend-YOUR-ID.onrender.com

TEST IT:
  https://authorprint-backend-YOUR-ID.onrender.com/api/health

You should see: {"status": "ok", "timestamp": ...}

THEN:
- Go back to Render Environment
- Set BACKEND_URL = your new URL
- Click Save (auto-redeploy)

✓ Backend is live!
